# Train-Rides-Dashboard
🚆 Train Rides Dashboard | Microsoft Excel  An interactive Excel dashboard designed to analyze train ride operations and performance. The dashboard provides insights into ticket sales, revenue generation, refunds, delays, ticket classes, time slots, and station performance using dynamic visualizations and slicers.

**Features**
1. Total Tickets Sold Analysis
2. Revenue Tracking by Month
3. Refund Analysis
4. Delay Category Distribution
5. Average Delay by Reason
6. Revenue by Ticket Class
7. Tickets by Time Slot
8. Top Departure Stations
9. Interactive Month and Day Filters
    
**Tools & Techniques**
1. Microsoft Excel
2. Pivot Tables
3. Pivot Charts
4. Slicers
5. Data Visualization
6. Dashboard Design
   
**Key Insights**
Monitor overall business performance through KPIs.
Identify peak ticket booking periods.
Analyze revenue trends and refund patterns.
Understand delay causes and operational efficiency.
Compare performance across stations and ticket classes.

**Skills**: Excel Dashboarding, Data Analysis, Data Visualization, Business Intelligence, Reporting, Pivot Tables, Pivot Charts.
